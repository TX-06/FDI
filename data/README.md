# Public Data Notes

This directory contains public numeric records and source-tracking files for the FDI manuscript.

## Included

- Numeric FDI outputs for controlled synthetic, simulated, rotation, waviness, spectral, and public VTON stress-test cases.
- Public image source inventory for VTON samples, including URLs and permission-status notes.
- General IQA comparison records where available.

## Excluded

- Third-party public VTON image crops.
- Source screenshots from project pages, papers, videos, or supplementary materials.
- Human-participant data, observer ratings, or perception-study records.
- Local manuscript-rendering outputs.

Local crop paths in released CSV/JSON files are replaced with `not_released_due_to_third_party_permissions`. Users who need to reproduce the public VTON stress test should consult the source inventory and verify permissions directly with the original sources.
